# Quiz básico en JavaScript

Proyecto simple de práctica que implementa un cuestionario interactivo usando HTML, CSS y JavaScript puro.

El objetivo del proyecto fue practicar manipulación del DOM (Document Object Model: estructura que representa el HTML y permite modificarlo con JavaScript), manejo de eventos y lógica básica de estado.

## Funcionalidades

* Mostrar preguntas por secciones
* Selección de respuestas con botones tipo radio
* Validación para evitar avanzar sin responder
* Cálculo de puntaje final
* Reinicio del cuestionario

## Tecnologías utilizadas

* HTML
* CSS
* JavaScript (Vanilla JS: JavaScript sin frameworks ni librerías)

## Objetivo del proyecto

Este proyecto fue desarrollado como ejercicio para practicar:

* Manipulación del DOM
* Eventos (`addEventListener`)
* Manejo de estado en variables
* Interacción básica con el usuario
